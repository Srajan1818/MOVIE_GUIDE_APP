# 🎬 Movie Guide App

A responsive **Movie Guide App** built with **HTML**, **CSS**, and **JavaScript** that allows you to search any movie and instantly get details like rating, release date, cast, and plot summary.

---




## ✨ Features

- 🔍 Search any movie by name in real time
- ⭐ Displays movie rating and review score
- 📅 Shows release date and movie duration
- 📖 Full movie plot summary
- 🎥 Movie poster with high quality image
- 🌐 Powered by OMDB API

---

## 🎯 Project Purpose

This project was created to strengthen the following concepts:

- 🌐 Fetching real time movie data from OMDB API
- 🔍 Implementing a live movie search functionality
- 🖼️ Dynamically rendering movie posters and details
- 📖 Displaying structured movie information like cast and plot
- 📱 Building a fully responsive and visually appealing UI

---

## 🛠️ Technologies Used

- **HTML5** – Structure and markup
- **CSS3** – Styling and responsive layout
- **JavaScript (ES6+)** – App logic and API integration
- **OMDB API** – For fetching real time movie data and details

## 📌 How To Use

1. Type any **movie name** in the search bar.
2. Click the **Search** button.
3. View the **movie poster**, **rating** and **release date**.
4. Read the full **plot summary** and **cast details**.
5. Explore more movies and discover new ones! 🍿
